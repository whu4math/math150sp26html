4.3 Logarithmic Functions

Motivation

\(g(x) = \dfrac{x-1}{2x+1}\) is one-to-one. We may use the “switch and solve” method to solve for its inverse.

\(f(x) = 2^x\) is also one-to-one. Can you solve for the inverse?

Logarithmic Function

We define the inverse of exponential functions as logarithmic functions.

\[a^y = x \iff \log_a x = y\]

In words, “a raised to the power of y is x” is equivalent to “log base a of x is y”.

Remarks:

  • Both functions have the same base \(a\) with \(a > 0\) and \(a \neq 1\).

  • The input of the exponential function (the exponent) is the output of the logarithmic function. So \(\log_a x\) is the power to which \(a\) is raised to get \(x\). Remember this: log gives us power.

  • The output of the exponential function is non-negative. So the input of the log function is non-negative.

Problem 2. Use exponential form to evaluate log.

  1. \(\log_2 32 = 5\) (Because \(2^5 = 32\))

  2. \(\log_{10} 1000 = 3\) (Because \(10^3 = 1000\))

Problem 3. Write in exponential form and solve for \(x\).

  1. \(\log_2 x = 3\)

    \(2^3 = x\)

    \(x = 8\)

  2. \(\log_3 \left(\frac{1}{27}\right) = x\)

    \(3^x = \dfrac{1}{27} = \left(\dfrac{1}{3}\right)^3 = (3)^{-3}\)

    \(x = -3\)

Special Logarithms

Common log (base 10): \(\log x = \log_{10} x\)

Natural log (base \(e\)): \(\ln x = \log_e x\)

Problem 4. Solve for \(x\).

  1. \(\ln x = -1\)

    \(e^{-1} = x\)

    \(x = \dfrac{1}{e}\)

  2. \(\log 0.001 = x\)

    \(10^x = 0.001 = 10^{-3}\)

    \(x = -3\)

Properties of log

General Common log Natural log
\(\log_a 1 = 0 \) \(\log 1 = 0\) \(\ln 1 = 0\)
\(\log_a a = 1\) \(\log 10 = 1\) \(\ln e = 1\)
\(\log_a a^x = x\) \(\log 10^x = x\) \(\ln e^x = x\)
\(a^{\log_a x} = x\) \(10^{\log x} = x\) \(e^{\ln x} = x\)

Problem 7. Find the domain.

  1. \(f(x) = \log_3(2 - 8x)\)

    Need: \(2 - 8x > 0\)

    \(2 > 8x\)

    \(\frac{2}{8} > x\)

    \(x < \frac{1}{4}\)

    Domain: \(\left(-\infty, \frac{1}{4}\right)\)

  2. \(g(x) = \ln(x - x^2)\)

    Need: \(x - x^2 > 0\)

    \(x(1 - x) > 0\)

    \(x(1-x)\) is a parabola in factored form with zeros at 0 and 1.

    From the graph we see it is positive when \(x\) in \((0, 1)\).

    Domain: \((0, 1)\)

Problem 8. Let’s solve this problem from the last section. The thickness of a piece of paper is about 0.1 mm. The thickness doubles each time as you fold it. If it is physically possible, after how many folds, the thickness will be equal to the distance between Earth and the Moon (384,400 km)?

This is similar to compound interest where the thickness doubles with each fold.

Let \(x\) = number of folds

Convert to same units: \(384,400 \text{ km} = 384,400 \times 10^6 \text{ mm}\)

\[\begin{align*} 0.1 \cdot 2^x &= 384,400 \times 10^6 \text{ mm} \\ 2^x &= 384,400 \times 10^7 \\ \log_2 (384,400 \times 10^7) &= x \\ x &\approx 41.8 \end{align*}\]

Therefore, it would take approximately 42 folds.
